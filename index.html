A friendly bot wasting peoples time and making the happy.

This automatic twitter bot tweets people it finds to start conversations, and send encouragmenets. It incorporates a lot of randomness, but a snap of the resulting conversations can be seen below.

The bot communicates via the twitter API and is built on the python implementation tweepy.
